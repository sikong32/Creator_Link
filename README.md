# 크리에이터 링크(CreatorLink)
팀원(서태덕,장석영,이윤,곽성유)
스토어(서태덕) 게시판(이윤) 회원관리(장석영) CSS(곽성유)

## 프로젝트 소개
- 프로젝트 실행 전 `TBL.zip` 압축 풀기 후 loader 형식으로 DB에 데이터 임포트
- 크롤링으로 방송 정보를 가져오기 때문에 크롤링하지 않을 시 방송 정보가 없을 수 있습니다.
